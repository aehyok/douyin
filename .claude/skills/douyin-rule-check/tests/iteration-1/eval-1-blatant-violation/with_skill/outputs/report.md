# 抖音/星图合规自检报告

## 总体评估
- 风险等级：**高**
- 违规点数量：6
- 发布建议：**直接拦截，必须修改后才能发布**。这一稿在 A、B 两大违规类别上几乎都踩了，属于星图预审会一次性驳回的典型样本。

## 详细问题

### 1. [位置：口播第 2 句 — "想要的姐妹评论区扣 1，我私聊给你们发链接"] — A2 引导用户私下联系
- 原文："想要的姐妹评论区扣 1，我私聊给你们发链接"
- 问题：典型的「评论区扣 1 + 私聊发链接」组合，明确把交易动作引导到抖音公开链路之外（私信/外链），属于引导脱离平台交易。
- 依据：A2 引导用户私下联系 /《巨量星图内容营销规范》引导脱离平台至其他渠道交易。
- 建议改写："这款面膜我放在下方购物车了，姐妹们点小黄车就能直接看到。"

### 2. [位置：口播第 2 句 — "加我微信「meili2025」"] — A1 引导联系方式联系
- 原文："加我微信「meili2025」还能再领 50 元红包"
- 问题：直接报出微信号，属于在视频里露出第三方联系方式，并叠加了"加微信领红包"的福利钩子，A1 + 福利诱导，确定违规。即便把"微信"写成"薇信""V"或者用 emoji 替代，按规范同样按违规处理。
- 依据：A1 引导联系方式联系 + 福利诱导动作。
- 建议改写：整句删除。如果想保留福利信息，改成："这次抖音店铺里有满减活动，具体优惠看商品详情页就行。"（把福利绑定到抖音站内交易，不出现微信、不出现"加我"动作）

### 3. [位置：口播第 3 句 — "下载我们家 app「美丽星球」注册就送一片体验装"] — B3 福利诱导下载 + B1 直接引导下载
- 原文："下载我们家 app「美丽星球」注册就送一片体验装"
- 问题："下载 app + 注册即送"是规范里点名的福利诱导下载典型话术，命中 B3；同时"下载 xx"本身也命中 B1。属于必拦截级别。
- 依据：B1 直接引导下载 / B3 福利诱导下载。
- 建议改写："这个产品在我们抖音店铺里也能直接下单，体验装活动看小黄车就好。"（去掉"下载 / 注册即送"的动作钩子，把福利信息留在抖音站内）

### 4. [位置：口播第 4 句 — "应用商店搜「美丽星球」就有"] — B1 直接引导下载
- 原文："应用商店搜「美丽星球」就有"
- 问题："应用商店搜 xx"是规范里逐字列出的违规表达，等同于命令式引导下载第三方 app。
- 依据：B1 直接引导下载（"应用商店搜 xx"）。
- 建议改写：整句删除。如果一定要提到自家品牌 app，可以改成纯介绍式："我们品牌也有自己的护肤小工具，姐妹们感兴趣可以了解一下「美丽星球」这个产品。"——只做客观介绍，不带"下载/搜/打开"动作。但更稳妥的做法是这一整句直接删掉，避开任何风险。

### 5. [位置：口播第 1 句 — "这款面膜真的是我用过最绝的，敷完第二天皮肤水到能滴下来"] — 化妆品功效极限词/夸大宣传（中风险）
- 原文："最绝的""水到能滴下来"
- 问题：虽然不属于 A/B 两大类引导外流，但属于化妆品类目里常见的"绝对化用语 + 夸大功效"组合，星图针对美妆护肤类目对这种表达有额外提示，容易被预审标"夸大宣传"。
- 依据：广告法对绝对化用语的限制 + 抖音化妆品行业规范。
- 建议改写："这款面膜我个人用着挺喜欢的，敷完第二天皮肤的水润感很明显。"（把"最绝""能滴下来"这种主观极限表述改成可感知的描述）

### 6. [位置：口播末尾 — "看完赶紧去冲"] — 低风险（注意）
- 原文："看完赶紧去冲"
- 问题：本身不算违规，但跟上面那些「下载」「搜」「加微信」连在一起，会被审核理解为整体催促站外动作。单独存在没问题，跟违规话术绑在一起会拉高整段被判违规的概率。
- 建议改写：保留也可以，建议改成更明确指向站内的动作，例如："喜欢的姐妹小黄车带走。"
