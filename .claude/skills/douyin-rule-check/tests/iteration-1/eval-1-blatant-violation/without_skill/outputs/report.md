# 抖音/星图护肤口播稿合规预审报告

## 总体结论

**当前版本：高概率被预审驳回，不建议直接投稿。**

这条不到 60 秒的口播里，几乎踩中了 2025 年抖音/星图最敏感的两类红线：**引导脱离平台交易（私域引流）** 和 **诱导下载/使用第三方 App**。同时还存在化妆品广告法层面的绝对化用语和功效夸大问题。

---

## 违规点逐条扫描

### 1.「评论区扣 1，我私聊给你们发链接」
- **风险等级：高**
- **风险类型**：引导脱离公域 / 私域引流
- **说明**：通过评论互动+私信发链接，是平台模型最容易识别的高危句式。命中即驳。

### 2.「加我微信「meili2025」还能再领 50 元红包」
- **风险等级：极高（基本必驳）**
- **风险类型**：明示站外联系方式 + 利益诱导外流
- **说明**：
  - 直接出现"微信"+ 具体微信号；
  - 用"50 元红包"作为利益点诱导加微；
  - 这是星图预审最硬的红线之一，文字、口播、字幕、画面任一形式出现均判违规，谐音/变体（"V 信""威❤️""徾信"）一样命中。

### 3.「下载我们家 app「美丽星球」注册就送一片体验装」
- **风险等级：极高**
- **风险类型**：诱导下载/使用第三方 App
- **说明**：
  - 明确"下载 App"动作指令；
  - "注册就送"再叠一层利益诱导；
  - 星图规范明令禁止视频以任何形式诱导跳转/下载非抖音生态的 App。

### 4.「应用商店搜「美丽星球」就有」
- **风险等级：高**
- **风险类型**：诱导下载（变体）
- **说明**：即便不说"下载"两个字，"应用商店搜 XXX"在审核模型里等价于诱导下载，照样命中。

### 5.「这款面膜真的是我用过最绝的」
- **风险等级：中**
- **风险类型**：广告法绝对化用语
- **说明**："最绝的""最+形容词"在化妆品广告里属于典型违禁词，需改为相对化表达。

### 6.「敷完第二天皮肤水到能滴下来」
- **风险等级：中**
- **风险类型**：化妆品功效夸大 / 虚假宣传嫌疑
- **说明**：化妆品不能用超出备案功效的过度可视化描述。"水到能滴下来"属于明显夸大。

### 7.「想要的姐妹…看完赶紧去冲」
- **风险等级：低**
- **风险类型**：催促下单话术（与上面违规组合后会被叠加判定为完整的"引流外流链条"）

---

## 能不能过

**不能。** 当前稿件至少包含 **2 个极高风险点**（微信号 + 下载自家 App），任一单独出现即足以驳回。这不是修修补补能过的，需要把整条引流链路从外流改回站内。

---

## 修改思路（先改逻辑，再改用词）

1. **所有交易/转化入口必须留在抖音站内**：用抖音小店、橱窗、商品链接、抖音群聊、企业号私信，不要让用户跳到微信或第三方 App。
2. **不要出现** 微信、QQ、加 V、扣扣、私信发链接、应用商店、下载、注册送 等关键词及其谐音变体。
3. **化妆品功效**：只描述使用感受，不夸效果，不用"最"。
4. **利益点照样可以给**，但只能挂在抖音生态内（购物车、店铺优惠券、粉丝群福利）。

## 改后版本（合规口播）

> 姐妹们，最近回购的这款面膜真的挺让我惊喜，上脸三分钟揭下来，皮肤是那种摸上去滑滑的、感觉喝饱水的状态，干皮敏感肌真的可以试试。
>
> 想入的姐妹直接点下方购物车的小黄车，店铺新粉关注就能领优惠券，叠加平台券到手价比我自己买还便宜。喜欢的话别忘了点个小红心，我后面会接着分享我的护肤好物，下次见。

### 这版做了什么
- 删除了微信号、红包外流、App 下载、应用商店搜索；
- "最绝的" → "挺让我惊喜"，去绝对化；
- "水到能滴下来" → "摸上去滑滑的、感觉喝饱水"，从夸大效果改成主观使用感受；
- 转化入口收回站内（小黄车 + 店铺券 + 平台券）；
- 利益点保留（新粉券、叠加券），但全部在抖音生态内闭环；
- 结尾收在站内互动（点赞 + 关注），不催促外流。
