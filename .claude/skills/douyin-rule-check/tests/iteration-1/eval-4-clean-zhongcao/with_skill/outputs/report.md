1. **"包装上有详细成分表可以自己看"**——把信息源指向产品本身（线下产品包装），而不是导流到任何站外平台或 app，是非常干净的合规表达。
2. **"我连续用了一周……"**——典型的分享式 / 体验式表达，符合官方 QA 明示的"种草式产品表达"白名单。
