# 抖音种草视频合规预审报告

## 整体结论

**风险等级：中等（建议修改后再发）**

整体看这条内容相对干净，没有引导外流、没有诱导下载第三方 app，两个最容易被预审驳回的红线没踩。但作为化妆品类目种草，里面有几处**功效宣称**和**成分浓度**表述，按照抖音对化妆品/广告内容的审核口径以及《广告法》《化妆品监督管理条例》的要求，存在被驳回或限流的风险。

---

## 逐条排查

### 一、引导脱离平台交易（站外/私域/线下）
**结论：未发现违规。**
- 口播和字幕都没有"加微信""私聊""x 元""到店""官网搜索""某宝某猫"等表述
- 没有"评论区扣 1""主页置顶""点头像看链接"等引流话术
- 未提及任何不明渠道或线下门店

### 二、诱导下载/使用第三方 app
**结论：未发现违规。**
- 未提及小红书、淘宝、得物、微信、微店等任何外部 app
- 没有"下载 xx 看完整测评""xx 平台首发"

### 三、化妆品广告功效宣称（**主要风险点**）

这是这条内容里最需要警惕的部分。审核参照国家药监局《化妆品功效宣称评价规范》和《广告法》第二十八条（虚假广告）。

**风险点 1：字幕「干皮亲妈」**
- 「亲妈」属于绝对化、夸大化表述，与「最强」「神器」「万能」同类，化妆品类目下高风险
- 抖音对化妆品广告里的"绝对化用语"和"暗示功效极致"的表达普遍会限流或驳回
- 建议改为：「干皮姐妹友好」「干皮日常用」「干皮可冲」

**风险点 2：「干皮的脸不紧绷了」「第二天起来气垫还是服帖的」**
- 这是**功效宣称**。化妆品宣称"保湿"允许，但"不紧绷""第二天还服帖"这种**对比性、效果承诺型**表述，化妆品广告里需要功效评价报告支撑
- 纯个人体验式表达相对宽松，但建议加"个人感受，因人而异"兜底
- 建议：保留个人体验框架，加一句"个人感受，肤质不同效果可能不同"，或字幕加一行"个人体验仅供参考"

**风险点 3：「连续用了一周」+ 效果描述**
- 暗示"7 天见效"，属于**时间承诺型功效宣称**，化妆品广告敏感
- 建议改为"用了一段时间"或不带时间表述

### 四、成分浓度标注（**次要风险点**）

**风险点 4：口播「加了 5% 的角鲨烷和透明质酸」 vs 字幕「角鲨烷 5%」「透明质酸 0.1%」**
- **口播歧义**：可能被理解为"两者都是 5%"，与字幕的 0.1% 不一致，审核可能判定为误导
- **合规要求**：化妆品广告标注具体成分浓度需与产品备案一致，否则属于虚假宣传
- 建议：
  - 口播改为「这款保湿霜里加了角鲨烷和透明质酸」，浓度只放字幕
  - 字幕浓度必须与产品官方/备案信息一致
  - 加兜底「具体成分以包装为准」

### 五、医疗/夸大用语扫描
**结论：基本干净。**
- 未出现「修复」「治疗」「祛痘」「美白」「淡斑」「抗衰」等医疗类或特殊功效宣称
- 「保湿」属于普通功效宣称，允许
- 「对成分敏感的姐妹买之前先看下」反而是加分项

### 六、其他
- 画面"博主面部特写 + 手持产品出镜"无明显违规
- 未出现未成年人、医疗场景、医院/白大褂等敏感元素

## 修改建议（直接可用版本）

**口播改写**：
> 「这款保湿霜里加了角鲨烷和透明质酸，质地不黏腻，我自己用下来干皮的脸没那么紧绷，晚上做完护肤第二天上妆也比较服帖。具体成分和浓度包装上都有，对成分敏感的姐妹买之前可以先看一下。个人感受，肤质不同体验可能不一样。」

**字幕改写**：
- 「角鲨烷 5%」→ 保留（前提：与备案一致）
- 「透明质酸 0.1%」→ 保留（前提：与备案一致）
- 「干皮亲妈」→ 改为「干皮日常」或「干皮友好」
- 新增小字：「个人体验仅供参考」

**画面**：维持原方案。

## 风险汇总

| 风险类别 | 状态 |
|---|---|
| 引导外流/私下交易 | 未发现违规 |
| 诱导下载第三方 app | 未发现违规 |
| 化妆品功效绝对化用语 | 有风险（干皮亲妈、效果承诺） |
| 成分浓度表述不一致 | 有风险（口播 5% vs 字幕 0.1%） |
| 医疗/夸大宣称 | 未发现违规 |
| 画面合规 | 未发现违规 |

**结论**：两个最致命的"外流"和"诱导下载"红线没踩，整体过审概率不低。但「干皮亲妈」和成分浓度口径不一致这两点很容易在预审被打回，建议按上面修改版本调整后再投稿。
