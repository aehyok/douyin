# 抖音视频文案合规预审报告

## 一、原始文案

> 「这个转场效果是我用剪映做的，剪映里有个「混合模式」的功能特别好用，调成『正片叠底』就能做出这种叠加效果。我在抖音上看到很多博主都在用这个，挺简单的，大家可以自己试试看。」

## 二、整体结论

**结论：基本合规，过审风险较低。但存在一处需要留意的边界点（提及「剪映」），建议根据账号性质判断是否优化。**

- 不构成「引导脱离平台交易」违规：没有私下联系、站外购买、线下渠道等内容。
- 是否构成「诱导下载第三方 App」需要分情况看：剪映是字节跳动旗下产品，与抖音同属一个生态，**单纯提及剪映作为创作工具，通常不会被判违规**；但在商单（巨量星图任务）场景下需要额外注意是否触达「品牌冲突/竞品」相关条款。

---

## 三、逐条风险扫描

### 1. 引导脱离平台交易（高优先级风险类目）

| 检查项 | 是否触发 | 说明 |
|---|---|---|
| 私下联系（加微信/QQ/留电话） | 否 | 文案无任何联系方式引导 |
| 站外平台跳转（淘宝/京东/小红书/微信/B站等） | 否 | 文案未提及任何外部交易或内容平台 |
| 线下店铺/到店消费引导 | 否 | 无线下场景 |
| 不明渠道（"私信我""评论扣1领"等） | 否 | 无任何私信、评论领取话术 |

**此类风险 = 0。**

### 2. 诱导下载/使用第三方 App（高优先级风险类目）

| 检查项 | 是否触发 | 说明 |
|---|---|---|
| 明确说"下载 XXX"/"去应用商店搜" | 否 | 文案无下载指令 |
| 提及竞品 App 并引导使用 | 部分相关 | 提到「剪映」并说「大家可以自己试试看」 |
| 教程明显依赖某个第三方 App 的功能 | 是，但是同生态产品 | 教程主体就是剪映的「混合模式」功能 |

**关键判定：剪映 ≠ 第三方竞品**

- 剪映（国内版）是字节跳动旗下产品，与抖音同属一个商业生态。
- 抖音/星图规范中所谓「诱导下载第三方 App」，主要针对**站外竞品平台**（其他短视频 App、电商 App、与剪映构成竞争的剪辑工具等）。
- 在抖音生态内，剪映通常被视为**官方推荐的创作工具**，平台在很多场景主动鼓励创作者使用剪映。
- 单纯展示「我用剪映做的」+ 教操作技巧，属于普通创作者教程内容，过审基本无碍。

**普通创作者非商单场景下，此风险 ≈ 0。**

### 3. 商单（巨量星图任务）场景下的额外风险

如果这条视频是接的**巨量星图商单**，需要额外注意：

| 检查项 | 风险等级 | 说明 |
|---|---|---|
| 品牌方是否为剪映竞品 | 高 | 如果品牌方是其他剪辑工具（Premiere、必剪、快影、CapCut 海外版等），安利剪映会触发「品牌冲突/竞品提及」 |
| 品牌方是否为剪映/字节系 | 低 | 如果是剪映或字节系商单，提及剪映完全合理 |
| 商单 brief 是否要求"画面不出现其他工具品牌" | 中 | 部分品牌方要求素材纯净，提及剪映可能不符合商务约定（这是商务条款而非平台规则） |

**非商单场景几乎无风险；商单场景需先确认品牌方与剪映的关系。**

### 4. 其他常见风险点扫描

| 检查项 | 是否触发 | 说明 |
|---|---|---|
| 极限词/绝对化用语 | 否 | 「特别好用」「挺简单」属于正常口语表达 |
| 虚假宣传/夸大效果 | 否 | 教程类客观陈述 |
| 未经证实的功效宣称 | 否 | 不涉及医疗、金融、教育效果等敏感领域 |
| 抄袭/搬运暗示 | 轻微提示 | 「我在抖音上看到很多博主都在用这个」不违规，但语义上略弱化原创性，对账号长期权重略有不利（属流量建议而非合规问题） |
| 低质引导（"评论扣1""三连"等） | 否 | 无 |

## 四、修改建议（可选优化，不是必须）

### 普通创作者直接发布
**当前文案可直接发，无需修改。**

### 想让文案更"安全 + 流量更友好"

1. **弱化「我在抖音上看到很多博主都在用」**
   不违规，但弱化了你的原创性，影响账号标签和流量分配。
   建议改为：「这是最近很火的一个转场玩法」或直接删掉这句。

2. **结尾 CTA 更明确（提升完播和互动）**
   「大家可以自己试试看」相对平淡，可改为「记得收藏，下次剪片能用上」——既不违规，又能拉收藏率。

### 商单场景

- 先确认品牌方是否是剪映或剪映关联方。
- 如品牌方是其他剪辑工具/竞品，**移除「剪映」相关内容**，改用通用表述（如"我用的剪辑软件里"），或换成品牌方指定工具。
- 如商单 brief 明确要求"画面不出现其他品牌 LOGO/名称"，同样规避剪映品牌。

## 五、总评

| 维度 | 评分 | 说明 |
|---|---|---|
| 引导脱离平台交易 | 安全 | 零触发 |
| 诱导下载第三方 App | 安全（同生态） | 剪映为字节系产品，不构成"第三方竞品"诱导 |
| 极限词/虚假宣传 | 安全 | 无 |
| 抄袭/搬运风险 | 安全 | 不构成实质性搬运声明 |
| 商单场景额外风险 | 取决于品牌方 | 需确认是否剪映竞品 |

**总体结论：非商单场景下，这条文案过审风险很低，可以直接发。商单场景下，先核对品牌方再决定是否保留「剪映」字样。**
