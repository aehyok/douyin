# douyin-rule-check

抖音/巨量星图视频内容合规预审 skill。基于 2025 年 12 月更新的《巨量星图内容营销规范》和《抖音社区自律公约》，在视频投稿前对脚本、口播、字幕、画面做违规自检，重点识别两大类风险：

- **A. 引导用户脱离平台至其他渠道交易**（私下联系、站外平台、线下店铺、不明渠道）
- **B. 引导下载/使用第三方软件**（直接下载、福利诱导、提供下载渠道）

星图自 2025-03-14 起在预审环节驳回涉及以上违规的视频。本 skill 帮你在投稿前找出会被驳的点，并给出可直接替换的合规表达。

规范原文：https://www.douyin.com/rule/policy#heading-13

## 这是什么

一个 Claude Code / Claude.ai 可加载的 skill。给它一段口播稿、字幕、分镜或画面描述，它会输出一份合规自检报告：

```
# 抖音/星图合规自检报告

## 总体评估
- 风险等级：高 / 中 / 低 / 通过
- 违规点数量：N（仅统计 A/B 类）
- 发布建议：直接拦截 / 修改后可发 / 可直接发布

## 详细问题（A 引导外流 / B 诱导下载）
[每条违规：位置 + 原文 + 子类型代号 + 建议改写]

## 通过项
[写得好的合规表达点出 1–3 处]

## 范围外提醒（可选）
[非 A/B 类但值得注意的问题，如广告法极限词]

## 备注
[整体提醒]
```

## 安装

把这个仓库 clone 或者 symlink 到 Claude Code 的 skills 目录：

```bash
# Claude Code (macOS)
git clone https://github.com/Jingyi-Wu-Richael/douyin-rule-check.git \
  ~/.claude/skills/douyin-rule-check
```

或者如果你已经在本地有这个 repo，做个软链接：

```bash
ln -s /path/to/douyin-rule-check ~/.claude/skills/douyin-rule-check
```

## 怎么用

直接在对话里贴口播/脚本/画面描述就行：

```
我准备发一条抖音视频，帮我看下口播稿有没有违规：
"姐妹们这款面膜真的绝绝子，想要的扣 1 我私聊发链接，加我微信「meili2025」还能领 50 元红包..."
```

skill 会自动触发并产出报告。

触发关键词包括：合规检查、合规自检、抖音规范、星图预审、过审检查、会不会被驳回、有没有违规、引导外流、诱导下载、视频/文案合规等。即使只说"帮我看下有没有问题"，只要上下文涉及抖音或星图发布，也会触发。

**不适用于**：微信视频号、小红书、B站、快手等其他平台——它们的规则不一样。

## 设计要点

skill 的判断有几个核心心智：

- **分享式 vs 命令式**：动作主语是观众（违规）还是"我"（合规）？目的地在不在抖音内？这是判断绝大多数边缘 case 的唯一锚点。
- **抖音生态边界**：剪映、抖音商城、抖店、巨量星图等是抖音生态内产品，引导使用允许。淘宝、微信、小红书、独立 app 都属于站外。
- **scope 严格收窄**：化妆品功效宣称、广告法极限词、抄袭、商单冲突等问题不进主违规列表——它们不是星图本次新规驳回项，会让用户在合规视频上做无用修改。
- **画面 + 口播合并判定**：口播再干净，画面里只要有二维码/箭头/联系方式特写也会被驳。

## 测试结果

iteration-1 用 4 个典型 case 做了 with-skill vs baseline 对比（baseline = 同样一个模型不挂 skill 直接答）：

| Case | with skill | baseline | 关键差异 |
|---|---|---|---|
| 1 明显违规（微信号+下载app+扣1私聊） | 7/7 | 7/7 | 都识别得出，with skill 用一半时间 |
| 2 画面违规（口播干净+二维码+箭头+扫码字幕） | 7/7 | 6/7 | baseline 给了**错误改法**（"应用商店搜 XX 是允许的"——规范里这恰恰是 B1 违规） |
| 3 边缘合规（剪映分享式表达） | 4/4 | 2/4 | baseline 把视频抄袭/商单冲突等无关问题混进来 |
| 4 纯种草（无任何引导） | 3/3 | 0/3 | baseline 把"干皮亲妈""连续用了一周"等广告法范围的边缘表述误判，让用户做无用修改 |

**通过率：100% vs 58.9%（+41.1%）**

详细报告见 `tests/iteration-1/`。
