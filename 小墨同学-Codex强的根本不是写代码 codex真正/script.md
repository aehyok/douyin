## 口播正文（AI少年口播版，可直接念 / 喂 TTS）

很多人以为 Codex 就是个写代码的工具，那你可能小看它了。它真正狠的地方根本不在写代码——而是背后这一整套技能，能把你的活几乎整个承包下来。我挑几个最实用的，一分钟给你过一遍。

大家好，我是 AI少年。

第一组，**打工最刚需的办公三件套**：Document、Spreadsheet、Presentation——直接对应 Word、Excel、PPT，文档、表格、演示材料让它帮你生成和修改，含金量不用我多说。

第二组，**想自己做网站和应用**，一条龙：**GitHub** 托管代码，**Vercel** 负责一键部署上线，**Supabase** 帮你搞定后端和数据库——后端代码一行都不用写，小白也能快速上手。

第三组，**纯代码做视频和素材**，两个工具：**hyperframes** 用 HTML 出画面，**Remotion** 用 React 出画面，都是写代码生成视频，效果非常稳。

第四组，**让它真的动起来**：**computer-use** 能直接操作你的电脑，**Chrome** 这一类能接管你的浏览器、获取信息、替你操作网站。到这一步，它就不只是一个会聊天的 AI，是真能替你干活的"手"了。

最后压轴的两个，**管技能的技能**：**skill-creator** 帮你创建、管理属于自己的技能；**skill-installer** 帮你一键把别人做好的技能拿过来就用。

办公、写代码、剪视频、操作电脑——全方位覆盖。"AI 帮你接管工作"，到这儿真的不只是一句口号了。

AI 的事，听 AI少年 说，关注我，咱们下期接着聊。
